# Password-Strength-Analyzer
A simple Python-based Password Strength Analyzer that checks password security based on:

- Password Length
- Uppercase Letters
- Lowercase Letters
- Numbers
- Special Characters

## Features

- Detects Weak, Medium, and Strong Passwords
- Provides suggestions for improving password strength
- Easy to use
- Built using Python

## Requirements

- Python 3.8+

No external libraries are required.

## Run the Project

```bash
python password_analyzer.py
```

## Example

Input:

Raj123

Output:

Password Strength: Medium

Suggestions:
- Use at least 8 characters
- Add at least one special character
